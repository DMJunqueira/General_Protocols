# This is a repository for LaBEVir protocols:





### Augur
Perform downsampling (subsampling) of a set of viral sequences based on user-defined criteria (e.g., collection date, geographic region, host), reducing the number of sequences in a balanced and reproducible manner while preserving the temporal and geographic representativeness of the dataset.

### CliqueSNV
Reconstructs viral quasispecies present in next-generation sequencing (NGS) data.

### ClusterPicker
Identifies clades that have an intraspecific genetic distance below a set threshold and selects only clades with a given node support.

### FigTree
Visualize, edit, and export phylogenetic trees.

### IQTree
Reconstruct phylogenetic trees using the Maximum Likelihood (ML) method. In addition, IQ-TREE can automatically select the best-fit evolutionary model and estimate branch statistical support.

### LSD2
Infer divergence dates (evolutionary timescale) in a phylogeny using the least-squares method implemented in LSD (Least-Squares Dating).

### MAFFT
Perform multiple sequence alignment of nucleotide or amino acid sequences.

### Miniconda
Install and configure Miniconda for managing environments and packages in computational analyses.

### PARNAS
Perform downsampling of phylogenetic trees in an objective and reproducible manner by selecting a minimal set of representative taxa that preserves the full genetic diversity within a defined sequence divergence threshold.

### RDP5
Identify recombinant sequences, their major and minor parent(s), and the corresponding recombination breakpoints.

### Tempest
Explore the temporal signal of sequence datasets, preferably with varying sampling dates.

### TreeTime
Infer the timescale of a phylogeny by estimating dates of common ancestors and evolutionary rates from sequence data and metadata.
